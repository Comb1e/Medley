# User Prompt
Write a Python project on Windows that requires implementing:
1. All outputs from this python project such as txt files should be placed in the output folder, and all inputs are placed in the videos folder
2. Check if the video comes with subtitles, which refer to subtitles that match the audio content in the video rather than separate SRT files
3. If the video comes with subtitles, extract them and place them in the output folder as a txt file;
4. If the video does not come with subtitles, recognize the subtitles through the audio of the video and add subtitles to the video to be placed in the output folder. Similarly, the subtitles need to be extracted separately and saved as a txt file;
5. The video format needs to support multiple common formats such as MP4
