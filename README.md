# 🚀 Quick Start
## Prequisites

1. Clone
```powershell
git clone https://github.com
cd
```
2. Setup environment
```powershell
pip install -r requirements.txt
```
3. Configure
```powershell
cp .env.example .env.local
```

Fill in varies under Essential
```powershell
#Essential
DASHSCOPE_API_KEY=...

BASE_AGENT_NAME=...
PROMPT_AGENT_NAME=...
CODING_MODEL_NAME=...
```
